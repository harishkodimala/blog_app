# Frontend - Blog App

Frontend for the MERN Blog Application built using React.js.

---

# 🚀 Features

- Responsive UI
- User Authentication
- Protected Routes
- Create & Manage Articles
- Dynamic Routing
- Axios API Integration
- Loading & Error States

---

# 🛠 Tech Stack

- React.js
- React Router DOM
- Axios
- CSS / Bootstrap

---

# 📂 Folder Structure

```txt
src/
│
├── components/
├── pages/
├── context/
├── assets/
├── App.jsx
└── main.jsx
```

---

# ⚡ Installation

```bash
npm install
```

---

# ▶ Run Development Server

```bash
npm run dev
```

Runs on:

```txt
http://localhost:5173
```

---

# 🔗 Backend Connection

Update API base URL inside:

```txt
src/config.js
```

Example:

```js
export const BASE_URL = "http://localhost:5000";
```
